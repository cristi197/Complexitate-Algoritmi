char[] vs string — două abordări

char[] (C-style)

Vector de caractere terminat cu '\0'. Funcții din <cstring>: strlen, strcpy, strcat, strcmp, strchr, strstr.

char s[201] = "salut";
string (C++)

Clasă C++, dimensiune dinamică. Include operatori: +, ==, []. Metodele: .length(), .substr(), .find(), .erase().

string s = "salut"; 
// char[] — declarare și inițializare
char s[201];                  // maxim 200 caractere + '\0'
char t[] = "informatica";     // dimensiune dedusă: 12

// string — declarare și inițializare
#include <string>
string str = "informatica";
string str2(5, 'a');            // "aaaaa"
La BAC, ambele variante sunt acceptate. Dacă se cere explicit „șir de caractere" sau lucrați cu char[]-uri, folosiți funcțiile din <cstring>.

Citirea șirurilor de caractere

char s[201];
string str;

// cin >> — citește până la primul spațiu/newline
cin >> s;         // "bac" (fără spații)
cin >> str;

// cin.getline — citește o linie întreagă (cu spații) — pentru char[]
cin.getline(s, 201);

// getline — citește o linie întreagă — pentru string
getline(cin, str);

// Atenție: dacă aveți cin >> n înainte de getline, consumați newline-ul!
int n;
cin >> n;
cin.ignore();          // sau cin.ignore(INT_MAX, '\n');
getline(cin, str);     // acum funcționează corect
Eroare frecventă la BAC: Dacă citiți un număr cu cin >> n și apoi un șir cu getline, fără cin.ignore(), getline citește linia goală rămasă!

Prelucrarea șirurilor — algoritmi frecvenți

Verificare palindrom

bool estePalindrom(string s) {
    int n = s.length();
    for (int i = 0; i < n/2; i++)
        if (s[i] != s[n-1-i]) return false;
    return true;
}
// "radar" → true, "bac" → false

Verificare anagramă

bool esteAnagrama(string a, string b) {
    if (a.length() != b.length()) return false;
    int freq[256] = {0};
    for (char c : a) freq[(unsigned char)c]++;
    for (char c : b) freq[(unsigned char)c]--;
    for (int i = 0; i < 256; i++)
        if (freq[i] != 0) return false;
    return true;
}
// "listen" și "silent" → true

Cifru Caesar (cifru cu deplasare)

string criptCaesar(string s, int k) {
    for (char &c : s) {
        if (c >= 'a' && c <= 'z')
            c = 'a' + (c - 'a' + k) % 26;
        else if (c >= 'A' && c <= 'Z')
            c = 'A' + (c - 'A' + k) % 26;
    }
    return s;
}
// criptCaesar("abc", 3) → "def"

Conversii număr ↔ șir

#include <string>   // to_string
#include <cstdlib>  // atoi, atof

// Număr → șir
string s = to_string(42);      // "42"
string pi = to_string(3.14);   // "3.140000"

// Șir → număr
int    n = stoi("123");        // 123
long   l = stol("999999");    // 999999
double d = stod("3.14");      // 3.14

// C-style: char[] → număr
int x = atoi("456");           // 456
double y = atof("3.14");       // 3.14
Sfat BAC: to_string() și stoi() funcționează pe compilatoare moderne C++11 și mai noi. La olimpiade se pot folosi și soluțiile manuale cu cifre.

Extragere cifre dintr-un număr (manual)

int nr = 12345;
char cifre[20];
int len = 0;
while (nr > 0) {
    cifre[len++] = '0' + nr % 10;
    nr /= 10;
}
cifre[len] = '\0';
// Inversăm: "54321" → "12345"
for (int i=0; i<len/2; i++) swap(cifre[i], cifre[len-1-i]);

Probleme tip BAC — șiruri de caractere

Problemă 1 (BAC 2022): Citiți un șir de caractere și afișați cel mai lung cuvânt.
Soluție: 
string linie, cuvantCurent, maxCuvant;
getline(cin, linie);
for (char c : linie) {
    if (c != ' ') cuvantCurent += c;
    else {
        if (cuvantCurent.size() > maxCuvant.size())
            maxCuvant = cuvantCurent;
        cuvantCurent.clear();
    }
}
if (cuvantCurent.size() > maxCuvant.size()) maxCuvant = cuvantCurent;
cout << maxCuvant;
Problemă 2: Dați un șir de caractere format din litere mici. Înlocuiți toate vocalele cu '*'.
Soluție: 
string vocale = "aeiou";
for (char &c : s)
    if (vocale.find(c) != string::npos)
        c = '*';
Problemă 3: Dată o propoziție, afișați cuvintele în ordine inversă.
Soluție: 
#include <sstream>
#include <vector>
string linie; getline(cin, linie);
stringstream ss(linie);
vector<string> cuvinte;
string w;
while (ss >> w) cuvinte.push_back(w);
for (int i = cuvinte.size()-1; i >= 0; i--)
    cout << cuvinte[i] << " ";
